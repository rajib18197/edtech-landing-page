Flexbox is a set of CSS properties for building 1 dimensional layouts. whereas Grid is for building 2 dimensional layouts. Flexbox makes it easy to automatically align items both horizontally and vertically. Grid makes it very easy for building more complex layout. grid allows us to write less nested html and easy to read css and flexbox allows us to write a little bit more nested html.

Bootstrap is a CSS Framework and Tailwind CSS is a utility first Components. Tailwind Provides varius of classes to style a certain component whereas bootstarp provides full-fledge component. So we can say Tailwind is much more customizable rather than bootstrap

Box Model defines how elements are displayed on a webpage and how their dimensions are calculated. According to CSS Specification, each and every element on a webpage can be seen as a rectangle box. Each box's dimensions are calculated via the box model. Box model has several components such as Content, a border, padding which is the transparent area around box and margin which is the space between boxes.
